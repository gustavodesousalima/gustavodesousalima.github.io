Habilidades

Aqui estão as principais tecnologias que utilizo em meus projetos:

  • HTML

    Utilizo HTML para estruturar o conteúdo das páginas da web, garantindo uma base sólida para o desenvolvimento.

  • CSS

    Tenho experiência em CSS para criar estilos e personalizar o visual dos elementos, tornando as interfaces atraentes e esteticamente agradáveis.

  • JavaScript

    Utilizo JavaScript para adicionar interatividade e funcionalidades dinâmicas aos projetos, proporcionando uma experiência mais envolvente aos usuários

  • React

    Utilizo o React para criar componentes modulares e reutilizáveis, que facilitam a construção de interfaces complexas.

  • NodeJS

    Utilizo o Node.js para construir aplicativos web do lado do servidor.Ele permite que eu desenvolva aplicativos usando JavaScript tanto no cliente quanto no servidor, facilitando a sincronização entre as duas partes.

  • Git

    Utilizo sistemas de controle de versão, como Git, para gerenciar o código-fonte dos projetos, facilitando o trabalho colaborativo e o rastreamento de alterações.

  • Mysql

    Tenho conhecimento em integração com bancos de dados no Node.js. Posso utilizar bancos de dados relacionais como o MySQL.

  • Scrum

    Tenho conhecimentos sobre a metodologia Scrum no desenvolvimento de projetos.
